# データベースマップ (仮称)

**公開URL**: [https://komageogra-webmaps.vercel.app/](https://komageogra-webmaps.vercel.app/)  
*(※全データの閲覧にはDiscordサーバーでのメンバー認証が必要です)*

## 使用技術 (Tech Stack)
- **フロントエンド**: Next.js (React), TypeScript
- **マップ描画**: MapLibre GL JS
- **データパース**: @tmcw/togeojson (KML/GPX -> GeoJSON変換)
- **バックエンド / DB**: Supabase (PostgreSQL, PostGIS, Storage, Auth)
- **認証**: Discord OAuth
- **ホスティング**: Vercel
